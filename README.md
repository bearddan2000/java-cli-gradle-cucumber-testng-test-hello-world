# java-cli-gradle-cucumber-testng-test-hello-world

## Description
A POC for gradle app using TestNG.
Uses cucumber plugin that publishes
report online.

## Tech stack
- java
- gradle
  - testng
  - cucumber

## Docker stack
- gradle:jdk11

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
